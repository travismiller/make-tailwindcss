# make-tailwindcss

A demo template for using [tailwindcss](https://tailwindcss.com/) with a simple Makefile.

## Requirements

- Node (>= 10.15.3)

## Usage

Straight compilation of `dist/styles.css`.

```console
$ make
```

Continuosly watch for changes and compile `dist/styles.css`.

```console
$ make watch
```
